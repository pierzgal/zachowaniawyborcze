3.2 Apportionment methods (metody podziału proporcjonalnego)

Drugi typ metod wyborczych (metody proporcjonalne) odnosi się do postulatu zachowania tzw. proporcjonalności wyborów. Otóż, podział mandatów między komitety wyborcze powinien jak najwierniej odzwierciedlać poparcie uzyskane przez poszczególne “ugrupowania”.

3.3 Apportionment methods (2)

Zasada proporcjonalności może być wyrażona następująco:

\[\frac{s_i}{S} = \frac{v_i}{V} \forall i\]

gdzie:

\(v_i\) = liczba wyborców głosującyh na i-ty komitet wyborczy;

\(s_i\) = liczba mandatów przyporządowana i−temu komitetowi wyborczemu;

\(TV\) lub \(V\) = sumaryczna liczba oddanych głosów ważnych w okręgu;

\(TS\) lub \(S\) = liczba dostępnych do podziału mandatów w okręgu.

3.4 Apportionment methods (3)

Przekształcając proporcję otrzymujemy:

\[s_i = v_i \frac{S}{V} \text{ lub równoważnie } s_i = \frac{v_i}{\frac{V}{S}} = \frac{v_i}{d}\]

\(d = \frac{V}{S}\) jest nazywany ilorazem wyborczym.

Zakładając, że \(S = 10, V = 100, v_1 = 40\).

Wtedy: \[\frac{\text{?}}{10} = \frac{40}{100}\]

  • Oczywiście, “?” = 4.

3.5 Standard quota

\[q_i = \frac{v_i}{\frac{V}{S}}\]

The standard quota i-tej partii (\(q_i\)) jest frakcją mandatów, którą uzyskałaby i-ta partia w okręgu, gdyby było możliwe przyporządkowanie jej “ułamkowych” mandatów.

3.6 Apportionment methods (4)

Proporcja to równość dwóch stosunków, np.:

\[\frac{1}{2} = \frac{2}{4}\]

1-out-of-2 = 2-out-of-4

Gdy stosunki (ułamki) są sobie równe, to są do siebie proporcjonalne.

3.10 Hamilton (Hare) Method

Podstawową zmienną różnicującą formuły reprezentacji proporcjonalnej jest metoda zaokrąglania reszt.

  • Metoda Hamiltona jest najprostrzym rozwiązaniem problemu zaokrąglenia reszt.

3.11 Hamilton (Hare) Method (2)

3.12 D’Hondt method

Remark: the result of D’Hondt apportionment \(\equiv\) the result of Jefferson apportionment

  • In D’Hondt (DH) algorithm, we use the following formula to distribute seats:

\[ quotient_{DH} = \frac{v_i}{s^*_i + 1} \]

where: \(v_i\) is the total number of votes that \(i-\)th party received and \(s^*_i\) is the number of seats that a party has been allocated so far, initially 0 for all parties.

  • D’Hondt method requires (see the example in: https://docs.google.com/spreadsheets/d/1QslIHty8AT_R7nLXiliYg4QQE2f0b7WqjWaG-eKFziw/edit?usp=sharing) the total numbers of votes received by parties to be divided first by one, then by two, then by three, and so on as necessary. The resulting quotients are then ranked by size, with the order determining entitlement to the seats available.
